- 입점 음식점 또는 메뉴 수 늘리기
  
##### 다음 서비스 지역 선별하기

## 개선 방법
##### 1. 잠재적 사용자를 정착시키기 위한 방법
- 서비스 지역 확장
- 입점 매장, 메뉴 다양성 확장
- 리워드 및 출석 쿠폰 등 보상제도 개발

##### 2. 새로운 서비스 개발
- 1인 가구의 앱 사용 목적에 맞는 서비스 개발

ex) 저렴한 가격으로 식사하는 것이 목적 → 앱 사용자가 직접 배달 → "당근마켓" 같은 문화 발생 → 리워드 및 쿠폰 지급 → 목적 달성
- 자체 메뉴 개발 (메뉴 선호도 활용)
- 한 매장 묶음 배달이 아닌 여러 매장 묶음 배달 (매장 밀집도 활용)
